Ayuda - Guía de Usuario

Navegación Principal

Inicio
Página principal que muestra un carrusel de imágenes de Oviedo y las últimas noticias relacionadas con la ciudad.
Juego
Juego de memoria donde debes encontrar parejas de imágenes relacionadas con Oviedo. Haz clic en las tarjetas para revelarlas y encuentra todas las parejas.
Calendario
Muestra el calendario de eventos importantes en Oviedo. Utiliza el botón "Cargar carreras" para ver la información actualizada.
Rutas
Permite cargar y visualizar rutas turísticas de Oviedo en diferentes formatos:
  • KML: Para ver rutas en Google Maps
  • SVG: Para visualizar mapas vectoriales
  • XML: Para información detallada de las rutas
Viajes
Sección para planificar visitas a Oviedo, incluyendo información sobre alojamiento y transporte.
Meteorología
Consulta el pronóstico meteorológico de Oviedo para los próximos días, incluyendo temperatura, probabilidad de lluvia y condiciones climáticas.
Gastronomía
Explora la rica gastronomía de Oviedo, con información sobre platos típicos, precios y experiencias culinarias recomendadas.

Funcionalidades Especiales

  • Carrusel de Imágenes: Usa los botones de navegación para ver diferentes imágenes de Oviedo.
  • Noticias en Tiempo Real: Las noticias se actualizan automáticamente para mostrarte la información más reciente.
  • Mapas Interactivos: Carga tus propios archivos KML para visualizar rutas personalizadas.
  • Pronóstico del Tiempo: Datos meteorológicos actualizados diariamente.

Consejos de Uso

  1. La web es completamente responsive - se adapta a cualquier dispositivo.
  2. Los archivos KML deben estar en formato válido para su correcta visualización.
  3. El pronóstico meteorológico se actualiza automáticamente cada vez que visitas la página.
  4. Puedes acceder a cualquier sección desde el menú de navegación superior.
