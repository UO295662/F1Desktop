Ayuda - Guía de Usuario

Navegación Principal

Inicio
Página principal con información general sobre Oviedo, incluyendo carrusel de imágenes representativas de la ciudad y enlaces a las principales secciones del sitio.
Juego
Test de conocimientos sobre esta pagina web. Preguntas sobre los temas de los que trata la aplicación.
Rutas
Sección completa de rutas turísticas por Oviedo que incluye:
  • Carga de archivos: Sube archivos KML, SVG o XML para visualizar rutas personalizadas
  • Mapas interactivos: Visualización con OpenLayers y marcadores de puntos de interés
  • Información detallada: Descripción, coordenadas GPS, distancias y galerías multimedia de cada hito
Reservas
Sistema de reservas con formulario de login y registro para usuarios. Permite realizar reservas de servicios turísticos en Oviedo con autenticación segura.
Meteorología
Información meteorológica completa que incluye:
  • Temperatura actual y sensación térmica
  • Pronóstico para los próximos 5 días
  • Humedad, viento y presión atmosférica
  • Datos actualizados mediante API meteorológica
Gastronomía
Guía gastronómica completa de Oviedo con:
  • Platos típicos asturianos (fabada, cachopo, carbayones)
  • Tabla de precios de restaurantes locales
  • Experiencias culinarias recomendadas
  • Galería multimedia con fotos, videos y audios
  • Información sobre sidrerías y la cultura de la sidra

Funcionalidades Especiales

  • Carrusel de Imágenes: Navegación automática y manual con botones de control para ver diferentes vistas de Oviedo.
  • Carga de Archivos: Sube tus propios archivos KML, SVG o XML para visualizar rutas personalizadas en mapas interactivos.
  • Mapas con OpenLayers: Visualización avanzada de mapas con marcadores, popups informativos y controles de zoom.
  • Gráficos SVG Dinámicos: Visualización de datos meteorológicos y altimetrías de rutas mediante gráficos escalables.
  • Multimedia Responsivo: Imágenes adaptativas con diferentes resoluciones según el dispositivo (picture element).
  • Formularios Interactivos: Sistema de login, registro y reservas con validación en tiempo real.
  • Datos en Tiempo Real: Información meteorológica actualizada automáticamente.

Consejos de Uso

  1. Diseño Responsivo: La web se adapta automáticamente a móviles, tablets y escritorio.
  2. Archivos KML: Deben estar en formato válido con coordenadas correctas para visualización en mapas.
  3. Navegación: Usa las migas de pan para saber tu ubicación actual en el sitio.
  4. Accesibilidad: Todos los elementos multimedia incluyen texto alternativo >li>
  5. Rendimiento: Las imágenes usan lazy loading para mejorar la velocidad de carga.
  6. Compatibilidad: Probado en los principales navegadores modernos (Chrome, Firefox, Safari, Edge).

Solución de Problemas Comunes

El mapa no se carga
Verifica tu conexión a internet y que el archivo KML tenga el formato correcto. Refresca la página si es necesario.
Las imágenes tardan en cargar
Normal en conexiones lentas. Las imágenes usan lazy loading y se cargan progresivamente al hacer scroll.
Los datos meteorológicos no se actualizan
Los datos se obtienen de APIs externas. Si no se cargan, puede deberse a problemas temporales del servicio.
